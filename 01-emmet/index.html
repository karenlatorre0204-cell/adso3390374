01-Emmet

Plugin for faster HTML and css coding.

01-Child >

Creates child elements (nesting)

Example:

ul>li>a

Result:

<ul> <li><a href=""></a></li> </ul>

02-sibling +

Creates sibling elements

Example:

header+section+footer

Result:

<header></header> <section></section> <footer></footer>

03-Multiplication *

Repeats an element or group of elements multiple times

Example:

ul>li*3

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04-class .

Selects all elements that have the specified class.

Example:

div.contenedor

Result:

<div class="contenedor"></div>

05-classes ..

Creates multiple class atributes

Example:

div.clase1.clase2.clasen

Result:

<div class="clase1 clase2 clasen"></div>

06-Id#

Creates an ID attributes.

Example:

nav#menu

Result:

<nav id="menu"></nav>

07-Text{}

Inserts text content.

Example:

a{click Aqui}

Result:

<a href="">click Aqui</a>

08-Custom Attribute[]

Creates custom attributes.

Example:

a[href=www.google.com]

Result:

<a href="www.google.com"></a>

09-Item Numbering$

Adds consecutive numbers.

Example:

ul>li.num$*3

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10-Reverse Numbering@-

Inverts numbering order.

Example:

ul>li.num$@-*3

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11-Base Numbering@n

Defines a base number for sequences.

Example:

ul>li.num$@3*4

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climb Up^

Climb Up

Example:

section>article>div^p

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping()

Groups elements together.

Example:

(header>nav)(section>article)

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-LoremLorem

Generates placeholder text.

Example:

lorem

Result:

Lorem ipsum dolor sit amet.

15-HTML Documents html

Creates various HTML document types.

Example:

html:xs

Result:

<html:xs></html:xs>

16-Meta Tagsmeta

Generates meta tags.

Example:

meta:vp

Result:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

17-challenge Emmet

It generates an HTML header navigation bar (a responsive navbar) with the following structure:

Example:

nav.navbar.navbar-inverse>div.container>(div.navbar-header>(button[type="button"].navbar-toggle>span.sr-only{Toggle navigation}+span.icon-bar*3)+a.navbar-brand[href="index.php"]>img[src="imgs/logo.png" alt=""])+div.collapse>ul.nav.navbar-nav>(li>a.link[href="index.html"]{Inicio})+(li.active.link>a[href="login.html"]{Ingresar})+(li>a.link[href="registro.html"]{Registrarse})+(li>a.link[href="contacto.html"]{Contacto})

Result:
